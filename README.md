# 📷 RTN Scan QR

A **React Native Fabric Native Component** for scanning QR codes on **Android** and **iOS**.

> ✅ This package is documented as compatible with **React Native 0.81+**.

---

## ✨ Features

- 🔍 Real-time QR code scanning
- ⚛️ Built for React Native Fabric architecture
- 📱 Android + iOS native implementations
- 🧩 TypeScript interface for native component usage
- 🚀 Lightweight package structure for easy integration

---

## 📦 Package Information

- **Package name:** `rtn-scan-qr`
- **Current version:** `0.0.2`
- **License:** MIT
- **Description:** QR Code scanner Fabric Native Component for React Native 0.81

---

## 🧱 Project Structure

```text
js/         -> TypeScript native component interface
android/    -> Android native implementation (Kotlin/Java)
ios/        -> iOS native implementation (Objective-C++)
```

---

## 🛠️ Installation

```bash
npm install rtn-scan-qr
```

or

```bash
yarn add rtn-scan-qr
```

---

## 🚀 Basic Usage

```tsx
import RTNScanQrNativeComponent from 'rtn-scan-qr';

export default function App() {
  return <RTNScanQrNativeComponent style={{ flex: 1 }} />;
}
```

---

## ✅ React Native Compatibility

This library is maintained to work with:

- **React Native 0.81 and newer**

If you are using an older React Native version, behavior may vary and additional native adjustments may be required.
